# 기능 요구사항
- 도서 검색 기능
  - 도서 제목으로 도서를 검색하는 기능 제공
  - 검색 결과는 페이징 형태로 제공
- 도서 검색 통계 기능
  - 사용자들이 검색한 도서에 대한 검색어 횟수 제공
  - 사용자들이 검색한 도서 순위를 상위 5개를 제공
